REFINED PROMPT FOR CAMELCASE FUNCTION:

Write a JavaScript function toCamelCase that converts strings to camelCase with comprehensive error handling and edge case management.

Requirements:
- Convert strings from snake_case, kebab-case, PascalCase, and SCREAMING_SNAKE_CASE to camelCase
- Include error handling for invalid inputs (non-string types, null, undefined)
- Throw descriptive error messages for invalid inputs
- Handle edge cases like empty strings, strings with multiple delimiters in a row, leading/trailing delimiters
- Handle strings that are already in camelCase
- Preserve single-word strings in lowercase

Examples:
- toCamelCase('first_name') should return 'firstName'
- toCamelCase('user-id') should return 'userId'
- toCamelCase('SCREEN_NAME') should return 'screenName'
- toCamelCase('mobile--number') should return 'mobileNumber'
- toCamelCase('_leading_underscore') should return 'leadingUnderscore'
- toCamelCase('') should return ''
- toCamelCase(null) should throw an error: "Input must be a string"
- toCamelCase(123) should throw an error: "Input must be a string"
- toCamelCase('alreadyCamelCase') should return 'alreadyCamelCase'
